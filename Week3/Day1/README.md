## 🔢 Numeric Functions Covered

---

### 🔹 ABS (Absolute Value)

Used to convert negative values into positive values.

✔ Common use: Error correction, financial adjustments

---

### 🔹 ROUND

Rounds numeric values to the nearest integer or decimal places.

✔ Common use: Payroll calculations and reporting

---

### 🔹 CEIL / CEILING

Rounds values **upward** to the nearest integer.

✔ Common use: Minimum billing or salary rounding

---

### 🔹 FLOOR

Rounds values **downward** to the nearest integer.

✔ Common use: Discount calculations or truncation logic

---

### 🔹 TRUNCATE

Cuts off decimal values without rounding.

✔ Common use: Precision control in financial systems

---

### 🔹 MOD

Returns remainder after division.

✔ Common use:

* Even/odd checks
* Pattern detection
* Scheduling logic

---

### 🔹 POWER / POW

Calculates exponent values.

✔ Common use:

* Growth calculations
* Financial projections
* Algorithmic computations

---

### 🔹 SQRT

Finds square root of a number.

✔ Common use:

* Statistical calculations
* Data normalization

---

### 🔹 SIGN

Returns:

* 1 for positive
* -1 for negative
* 0 for zero

✔ Common use:

* Data validation
* Integrity checks

---

### 🔹 RAND

Generates random numbers.

✔ Common use:

* Sampling
* Testing datasets
* Simulation systems

---

### 🔹 FORMAT

Formats numbers into readable string format.

✔ Common use:

* Salary display formatting
* Financial dashboards

---

### 🔹 GREATEST

Returns the largest value from multiple columns.

✔ Common use:

* Salary vs bonus comparison
* Maximum value extraction

---

### 🔹 LEAST

Returns the smallest value from multiple columns.

✔ Common use:

* Minimum salary detection
* Risk analysis
