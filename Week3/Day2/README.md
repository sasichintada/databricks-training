## 🔹 RANK() Queries

Implemented ranking operations with ranking gaps.

Examples:

* Employee salary ranking
* Department-wise ranking
* Order amount ranking
* City-wise order ranking
* Name-based ranking

Performed continuous ranking without gaps.

Examples:

* City-based order analysis
